# Répertoire des chants

Bienvenue dans le répertoire de la chorale **Siloé**. Vous trouverez ici toutes les paroles de nos chants de louange et d'adoration.

## Comment naviguer

Utilisez la barre latérale pour parcourir les chants par nom. Chaque page contient :
- Les **paroles complètes** du chant
- L'**auteur / compositeur** (quand disponible)
- Des **notes** éventuelles (tonalité, tempo, etc.)
